Syntax

Headings

Use the # sign for headlines:

 # Headline level 1

## Headline level 2

Lists

Use asterisks for bulleted lists:

 * Item
 * Item
 * Item

And digits for numbered lists:

1. First item
2. Second item
3. Third item

Blockquotes

Use email-style quotes:

> Standard old quote
>
> > A quote, quoted

Links

Bracket to link’s desired words and wrap the URL in parentheses:

[link text](url)

For more uses and examples, check the official Markdown syntax guide.

About

I love John Gruber’s Markdown web converter, affectionately termed Dingus. It’s a great tool for quickly converting Markdown text to copyable HTML, and I find myself using it constantly.

Mr. Gruber’s Dingus is, unfortunately, sometimes a bit slow because it posts content to a server for parsing. I recently began learning about JavaScript regular expressions, so as an exercise I thought I’d put together a client-side Markdown converter in hopes of making a faster convert and learning something new. I call it Capture.

Note

This tool has, invariably, some “n00b” errors. Please, please email me if you find a problem: cory@swashcap.com. The project is on the GitHub so, if you’re feeling energetic, contribute!
